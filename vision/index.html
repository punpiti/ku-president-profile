Question Bank

ใช้สำหรับ browse คำถามตามหมวด ฝึกตอบแบบ thematic และหา framing ที่พร้อมใช้บนเวทีหรือในห้องสัมภาษณ์

  • เหมาะกับการซ้อมตอบเป็นรายคำถาม
  • ดูภาพรวมคำถามหลักและคำถาม deep dive ได้เร็ว
  • ใช้เป็นฐานก่อนย่อคำตอบจริง
Open question-bank
Presentation Tool

Presentation Narrator

ใช้สำหรับทำ narration ต่อ slide, เกลา answer/version-to-say และตรวจหน้าพร้อมซ้อมจากไฟล์ markdown ที่ generate เป็น HTML

  • เหมาะกับการทำ slide-by-slide speaking notes
  • ดูสถานะ `ผ่านแล้ว` กับ `พร้อมปรับ` ได้จาก index
  • เชื่อมกับ workflow `markdown -> generated html` โดยตรง
