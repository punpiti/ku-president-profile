# Prompt For More Questions

ใช้ prompt นี้กับ AI อีกตัว เพื่อช่วยคิด “คำถามที่ควรถามเพิ่ม” โดยไม่ซ้ำกับคำถามที่เตรียมไปแล้ว

```text
คุณเป็นกรรมการสรรหาอธิการบดีมหาวิทยาลัย และเป็นผู้สังเกตการณ์ public hearing ที่ต้องช่วยคิด “คำถามเพิ่มเติม” สำหรับผู้สมัครคนหนึ่ง

เป้าหมาย:
- ช่วยตั้งคำถามใหม่ที่ยัง “น่าถามจริง”
- หลีกเลี่ยงคำถามที่ซ้ำหรือใกล้เคียงกับคำถามที่เตรียมไว้แล้ว
- เน้นคำถามที่ช่วยเปิด blind spot หรือ follow-up ที่คมขึ้น
- คิดทั้งคำถามจากมุมกรรมการสรรหา และคำถามจากมุมประชาคมมหาวิทยาลัย

บริบทของผู้สมัคร:
- ผู้สมัครมีจุดเด่นด้าน system thinking, data governance, digital transformation, curriculum/system design, cross-unit integration
- ผู้สมัครต้องใช้คำตอบได้ทั้งใน 2 เวที:
  1. public hearing / แสดงวิสัยทัศน์
  2. สัมภาษณ์ในห้องโดยกรรมการสรรหา

คำถามที่ “ถามไปแล้ว / เตรียมไว้แล้ว / ถือว่าครอบคลุมแล้ว” มีดังนี้

1. หากท่านได้รับเลือก อะไรคือ 3 เรื่องเร่งด่วนใน 1 ปีแรก?
2. การตั้งกลไกกำกับดูแล AI จะทำให้การทำงานช้าลงหรือไม่? และจะเพิ่มภาระบุคลากรหรือไม่?
3. ถ้าอาจารย์บางส่วนไม่ต้องการให้มีระบบข้อมูลกลาง เพราะกลัวถูกตรวจสอบ ท่านจะทำอย่างไรให้เกิดความไว้วางใจ?
4. ท่านมีแนวคิดเปลี่ยนแปลงหลายเรื่อง จะทำอย่างไรไม่ให้เกิดแรงต้านในปีแรก?
5. ท่านไม่เคยเป็นคณบดี ท่านจะบริหารมหาวิทยาลัยทั้งระบบได้อย่างไร?
6. แล้วท่านจะเข้าใจปัญหาระดับคณะได้อย่างไร ถ้าไม่เคยบริหารคณะ?
7. ทำไมสภาควรเลือกท่าน มากกว่าผู้สมัครอีก 2 คน?
8. ท่านจะทำให้ KU ขยับ ranking ได้อย่างไร ภายใต้ทรัพยากรจำกัด?
9. Digital University ภายใน 4 ปี จะทำจริงอย่างไร?
10. ถ้ารายได้จากการศึกษาไม่เพิ่ม ท่านจะหารายได้จากไหน?
11. ท่านจะจัดการงบขาดดุลหรือภาระบุคลากรอย่างไร?
12. หากคณบดีไม่เห็นด้วยกับนโยบายท่าน จะทำอย่างไร?
13. ท่านจะบริหารสภามหาวิทยาลัยอย่างไรให้ทำงานเชิงยุทธศาสตร์ ไม่ใช่เชิงจุกจิก?
14. จุดอ่อนของท่านคืออะไร?

ช่วยสรุปก่อนว่า “คำถามที่ครอบคลุมไปแล้ว” มี theme อะไรบ้าง เช่น strategy, finance, digital, governance, leadership, personal fit

จากนั้นช่วยเสนอ:
1. คำถามใหม่ 15 ข้อ ที่ “ไม่ซ้ำ” กับรายการข้างบน
2. แยกเป็น 3 กลุ่ม:
   - คำถามที่กรรมการสรรหาน่าจะถาม
   - คำถามที่ประชาคมมหาวิทยาลัยน่าจะถาม
   - คำถาม follow-up เชิงลึกที่อาจใช้สวนหลังคำตอบแรก
3. สำหรับแต่ละคำถาม ให้บอกสั้น ๆ ว่า:
   - ทำไมน่าถาม
   - มันวัดอะไร
   - มันต่างจากคำถามที่มีอยู่แล้วอย่างไร

ข้อสำคัญ:
- อย่าเสนอคำถามที่เป็นเพียงการ rephrase คำถามเดิม
- ถ้าคำถามใกล้กับของเดิม ให้บอกว่าใกล้กับข้อไหนและทำไมยังคุ้มจะถาม
- พยายามหาช่องว่างจริง เช่น blind spot, political risk, execution risk, coalition building, crisis handling, external stakeholder management, student issues, alumni, provincial campuses, union/HR conflict, legal/ethical risk, public communication

รูปแบบคำตอบที่ต้องการ:
1. Themes already covered
2. New high-value questions
3. Best 5 questions to ask first
```
