# Presentation Narrator Process Log

ไฟล์นี้ใช้จดการเปลี่ยนแปลงของกระบวนการทำงาน เพื่อไม่ให้วนกลับไปใช้กติกาเก่าโดยไม่รู้ตัว

## วิธีใช้

- ใช้ `README.md` เป็นคู่มือ “เวอร์ชันปัจจุบัน”
- ใช้ไฟล์นี้เป็นประวัติว่า workflow เปลี่ยนอะไรไปบ้าง
- ทุกครั้งที่เปลี่ยนกระบวนการ ควรจด 3 อย่าง:
  - เปลี่ยนอะไร
  - เปลี่ยนเพราะอะไร
  - ผลที่คาดหวังคืออะไร
