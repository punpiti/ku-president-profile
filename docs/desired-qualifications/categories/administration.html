หมวด 1

ลักษณะทั่วไปตามจำนวนอ้างอิง

จำนวนอ้างอิง = จำนวนครั้งที่รายการนั้นแทนเสียงตอบใน PDF โดยใช้เลขในวงเล็บท้ายข้อ ถ้าไม่มีเลขในวงเล็บนับเป็น 1

Sankey: เส้นทางการจัดกลุ่ม

แสดงเฉพาะรายการที่ถูกจัดเป็น “ลักษณะพึงประสงค์”: หมวด → ประเด็นลักษณะพึงประสงค์ → tag/topic → ข้อวิสัยทัศน์ที่สอดคล้อง โดยตัดการนับซ้ำด้วยการกระจายน้ำหนักของแต่ละข้อเมื่อเข้าหลาย tag และเรียงวิสัยทัศน์จาก บริบท, บทบาท, เป้าหมาย, กลยุทธ์ และแผนสี่ปี

ลักษณะพึงประสงค์ของอธิการบดี

ตัวเลขบนการ์ดคือจำนวนอ้างอิงของรายการที่เข้ากับลักษณะนั้น

ความสอดคล้องกับวิสัยทัศน์

เทียบทุกข้อในชุดบริบท บทบาท เป้าหมาย กลยุทธ์ และแผนสี่ปี จาก `KU/president/docs` โดยใช้ keyword matching เป็นรอบแรก รายการตัวอย่างยังอ้างอิงข้อและหน้าใน PDF ได้

ข้อกังวล

ตัวเลขบนการ์ดคือจำนวนอ้างอิงของรายการที่สะท้อนข้อกังวลนั้น

ประเด็นละเอียดอ่อน

แสดงเฉพาะรายการที่มีคำหรือบริบทเกี่ยวกับกลุ่มอำนาจ ผลประโยชน์ การแต่งตั้ง ความไม่เป็นธรรม หรือการเมืองภายใน ควรใช้เพื่อชี้ตำแหน่งหลักฐาน ไม่ใช่ขยายข้อกล่าวหา

ประเด็นอื่น

รายการที่อาจหลุดจากโจทย์คุณลักษณะของอธิการบดีในหมวดนี้ หรือเป็นรายละเอียดของหมวดอื่นโดยตรง

ตารางหลักฐานทั้งหมด

คอลัมน์ “จำนวนอ้างอิง” ใช้เลขในวงเล็บท้ายข้อ ถ้ามี; ถ้าไม่มีเลขในวงเล็บนับเป็น 1

อ้างอิง จำนวนอ้างอิง ข้อความ ประเภท tag/topic สอดคล้องกับวิสัยทัศน์ มั่นใจ
